# Personal Portfolio

Live site: https://YOUR-USERNAME.github.io

## How to customize
- `index.html`: update name, tagline, links, project content
- `styles.css`: adjust colors, spacing, layout
- `assets/avatar.png`: optional profile image
- `assets/Resume.pdf`: optional resume link

## Deploy
This repository is designed for GitHub Pages:
- Settings → Pages → Deploy from branch → main (root)
